Meet Me App

Meet Me App was created to allow users to view different Front-End developer events that take place all around the world.

The app was developed with the following five features in mind:

  • Filter by events
  • Show and hide event details
  • Specify the number of events
  • Use the app when offline
  • View a chart to see the number of upcoming events by city
